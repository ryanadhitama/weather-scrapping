# Weather Scrapping

This is my simple python weather scrapping from [7 day forecast](https://forecast.weather.gov/MapClick.php?lat=34.0536&lon=-118.2455#.YFhAmEgzZQI) website.

## How to use:
Install requirement :
```sh 
pip install -r requirements.txt 
```

Run application :
```sh 
python main.py
```
